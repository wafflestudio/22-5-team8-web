# 22-5 와챠피디아 Frontend

- [**와챠피디아(WachaPedia)**](https://d2vsqxcvld4zf7.cloudfront.net/)는 [**왓챠피디아(WatchaPedia)**](https://pedia.watcha.com/ko-KR)를 클론하는 클론 프로젝트입니다.
- 왓챠피디아는 본래 영화, 드라마, 웹툰 등 다양한 컨텐츠에 대한 자신의 평가 및 감상을 공유하는 서비스로, 이 프로젝트에서는 영화에 대해서만 구현하였습니다.
- 이 레포지토리는 Frontend 레포지토리로, Backend 내용은 [Backend 레포지토리](https://github.com/wafflestudio/22-5-team8-server)를 참고하시기 바랍니다.

## 배포

- Frontend prod 배포: https://d2vsqxcvld4zf7.cloudfront.net/
- Frontend dev 배포: https://d1xtmfti6ypide.cloudfront.net/
- 백엔드 배포: http://52.79.121.53

## 역할 분담

- 박인혁: 회원가입 및 로그인, 유저 프로필 페이지 구현
- 박종경: 랜딩 페이지, 검색, 컬렉션 관련 기능, 추천 영화 페이지, 소셜 로그인, 유저 프로필 관련 기능 구현
- 허태윤: 영화 페이지, 리뷰, 댓글, 취향 분석 페이지, 유저 차단 관련 기능 구현
